# Chapter 14: Capstone Project — The Autonomous Humanoid

> Content coming soon. This capstone brings together all modules into a complete autonomous humanoid system.

**Capstone objectives:**
- Integrate voice command → LLM planning → ROS 2 execution pipeline
- Implement VSLAM for environment mapping and navigation
- Object detection and manipulation using learned policies
- Deploy complete system in Isaac Sim
- Optional: transfer to physical robot (Unitree Go2 or G1)

**Deliverables:**
- A simulated humanoid robot that receives a voice command
- Plans a navigation path around obstacles
- Identifies and manipulates a target object
- Demonstrates full autonomy from perception to action
